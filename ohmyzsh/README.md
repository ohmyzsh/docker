# `ohmyzsh/ohmyzsh`

![Docker Image Size (latest semver)](https://img.shields.io/docker/image-size/ohmyzsh/ohmyzsh?sort=semver) ![Docker Pulls](https://img.shields.io/docker/pulls/ohmyzsh/ohmyzsh)

This image contains a version of the [`ohmyzsh/zsh` image](https://hub.docker.com/r/ohmyzsh/zsh)
and a version of Oh My Zsh that depends on the tag of the image pulled.

Example:

- `ohmyzsh/ohmyzsh:master`: contains the latest stable Zsh version as well
  as the latest commit of the Oh My Zsh repository.

- `ohmyzsh/ohmyzsh:5.9.1-master`: contains Zsh 5.9.1 as well as the latest
  commit of the Oh My Zsh repository.
